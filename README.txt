Monetize your site with Monetizer.

== Description ==

Monetizer is a revolutionary monetization platform which helps traffic sources, publishers & networks monetize their global inventory.

Monetizer works with more than 20 thousand Publishers and 8 thousand Advertisers.

Our WordPress Plugin currently offers several ad-formats - push notifications, in-text links, auto redirect and popups.

== Installation ==

1. [Sign up](https://www.monetizer.com/) for a Publisher account.
2. Add a domain name according to the [instructions](https://app.monetizer.com/academy/post/How-to-Add-a-Domain-to-Monetizer---Video-Tutorial-5).
3. Install the Monetizer plugin from the WordPress.org plugin directory or by uploading the Monetizer plugin folder to the `/wp-content/plugins/` directory.
4. Activate the Monetizer plugin through the "Plugins" menu in WordPress Admin.
5. Configure the plugin through the "Monetizer" menu in the WordPress Admin Dashboard.
